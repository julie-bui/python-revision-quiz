# python-revision-quiz
- Built a Python desktop application using Tkinter, MySQL, and JSON to create a multi-level revision quiz with easy, medium, and hard difficulties.

- Implemented user login/signup functionality, data persistence, and error handling for secure user management.

- Designed dynamic quiz logic, including score calculation, feedback messages, and restart/report options.

- Developed an interactive UI with buttons, labels, and radio buttons to provide an engaging user experience.
